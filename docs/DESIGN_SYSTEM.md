# Design System

UI components, tokens, and patterns for the Gollistic Healing platform.

## Principles

1. **Calm & trustworthy** — spacious layouts, soft contrast, healing-oriented aesthetics
2. **Accessible** — WCAG 2.1 AA minimum
3. **Consistent** — reuse tokens and components before one-offs
