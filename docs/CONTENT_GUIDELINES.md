# Content Guidelines

Standards for writing and structuring content on the Gollistic Healing platform.

## Purpose

Ensure all copy is clear, compassionate, on-brand, and optimized for readers and search engines.

## Page Structure

Every page should include:

1. **Title (H1)** — one per page, descriptive and unique
2. **Introduction** — 1–2 sentences: what this page offers and for whom
3. **Body** — scannable sections with H2/H3 headings
4. **Call to action** — one primary CTA per page where appropriate

## Writing Style

- Use second person (“you”) when addressing visitors
- Keep sentences short; one idea per paragraph
- Use bullet lists for steps, benefits, or features
- Define healing or wellness terms on first use when needed

See [Brand Guidelines](./BRAND_GUIDELINES.md) for voice and tone.

## Headings

| Level | Use for |
|-------|---------|
| H1 | Page title only |
| H2 | Major sections |
| H3 | Subsections within H2 |

Do not skip heading levels.

## CTAs

| Type | Example | Usage |
|------|---------|-------|
| Primary | Book a session | One per page section max |
| Secondary | Learn more | Supporting links |

## Images & Media

- Always include descriptive `alt` text
- Prefer original photography over generic stock
- Caption optional; use for context or attribution

## Legal & Compliance

- Include disclaimers where health or wellness claims appear
- Privacy policy and terms links in footer
- No guaranteed outcomes; use supportive, evidence-aware language

## Content Checklist (pre-publish)

- [ ] Unique title and meta description
- [ ] Correct heading hierarchy
- [ ] Links work; external links open appropriately
- [ ] Spelling and grammar reviewed
- [ ] Aligned with brand voice
- [ ] Mobile-readable length and formatting
