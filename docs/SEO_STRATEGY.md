# SEO Strategy

Search optimization plan for the Gollistic Healing platform.

## Objectives

- Rank for branded queries (“Gollistic Healing”, “Kanwaljit Singh”)
- Capture local and service-intent searches relevant to offerings
- Build authority through helpful, original content

## On-Page SEO

### Every page

- Unique `<title>` (50–60 characters)
- Meta description (150–160 characters)
- One H1 matching page intent
- Canonical URL
- Open Graph and Twitter card metadata

### Technical

- [ ] XML sitemap
- [ ] `robots.txt`
- [ ] Semantic HTML (`header`, `main`, `nav`, `footer`)
- [ ] Fast Core Web Vitals (LCP, INP, CLS)
- [ ] Mobile-friendly layout
- [ ] HTTPS only

## Structured Data

Consider JSON-LD for:

- `Organization` / `LocalBusiness` (if applicable)
- `WebSite` with `SearchAction` (if site search added)
- `Article` for blog posts
- `FAQPage` where FAQ content exists

## Content SEO

- Publish original articles aligned with audience questions
- Internal linking between services, blog, and contact
- Update cornerstone pages quarterly

See [Content Guidelines](./CONTENT_GUIDELINES.md).

- NAP consistency (Name, Address, Phone)
